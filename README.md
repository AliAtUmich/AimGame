# AimGame

A browser-based aim trainer built with HTML, CSS, and JavaScript.

## Features

- Adjustable mouse sensitivity and DPI
- Customizable crosshair styles and colors
- Real-time performance tracking (accuracy, hits)
- Responsive canvas layout
- Web-based and lightweight

## Live Demo

[Visit the Live Site](https://aliatumich.github.io/AimGame)

## Getting Started

Clone the repo and open `index.html` in your browser.
